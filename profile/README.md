# 🤰 MOMFIT

AI & IoT 기반 임산부 맞춤형 운동 및 건강 관리 서비스

MOMFIT은 임신 주차 기반 맞춤형 운동 추천과  
실시간 심박수 안전 모니터링을 통해  
임산부의 안전하고 체계적인 운동을 지원하는 통합 건강 관리 플랫폼입니다.

---

## 🚀 프로젝트 목표

- 임신 주차 및 분기 기반 맞춤형 운동 제공
- 운동 중 심박수 모니터링을 통한 위험 감지
- 주차별 건강 정보 및 가이드라인 제공
- 향후 LLM 기반 개인 맞춤 건강 코칭 확장

---

## ✨ 주요 기능

### 🤱 임신 관리
- 임신 정보 등록 (BMI 및 최대 허용 심박수 자동 계산)
- 임신 주차 및 분기 자동 계산
- 분기별 운동 가이드라인 제공
- 주차별 건강 정보 제공

### 🏃 운동 관리
- 운동 세션 시작 / 종료 / 일시정지 / 재개 / 중단
- 세션 기반 운동 기록 관리
- 운동 히스토리 조회

### ❤️ 심박수 모니터링
- IoT 기반 실시간 심박수 수집 구조 설계
- 최대 허용 심박수 초과 감지 로직
- 운동 강도 분석 기반 확장 가능 구조
